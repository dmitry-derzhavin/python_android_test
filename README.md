# Get project
```shellscript
git clone git@github.com:dmitry-derzhavin/python_android_test.git
cd python_android_test/
```

# Setup virtualenv
```shellscript
virtualenv venv3 -p python3
source venv3/bin/activate
pip install -r req.txt 
```

# Run
```shellscript
python main_simple.py
python main.py
python main_hard.py
```
